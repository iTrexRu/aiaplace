# Changelog Сессии

## [Test Release] - 2026-02-04

### Добавлено
*   **Документация**: Создана полная структура `docs/` (Context, Functions, Status, Todo, Rules).
*   **Статистика**: На главной странице добавлен дашборд (Всего контента, Сэкономлено денег, Статистика за сегодня).
*   **Интеграция AllPay**:
    *   Кнопка оплаты в профиле пользователя.
    *   Логика открытия оплаты во всплывающем окне (popup).
    *   Создан PHP-скрипт `webhook-allpay.php` для проксирования уведомлений от платежной системы в n8n.
*   **Тарифы**: В базу данных добавлены поля `tariff` и `tariff_expires_at`.
*   **UI Профиля**: Добавлено отображение статуса «PRO АКТИВИРОВАН» и даты окончания подписки.

### Изменено
*   **Оптимизация SQL**: Ветка `get_profile` в n8n переведена на новый SQL-запрос, который возвращает все данные пользователя и только актуальный онбординг за один вызов.
*   **README**: Актуализирован процесс деплоя и запуска.
*   **Rules**: Добавлен стайлгайд по CSS и соглашения по именованию.

### Исправлено
*   Исправлены синтаксические ошибки в SQL-запросах n8n (обработка кавычек и регулярные выражения).
*   Устранена ошибка `Module crypto is disallowed` в n8n путем перехода на подготовку строки подписи и использование встроенной ноды Crypto.
