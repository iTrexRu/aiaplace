# Статус Проекта

## Реализовано
*   [x] Базовая структура SPA (Router, API Client, Partials).
*   [x] Аутентификация (Вход, Регистрация, Выход, Восстановление пароля).
*   [x] Онбординг пользователя (Identity, Product, Audience, Style).
    *   [x] Экспресс-онбординг.
    *   [x] Скрытие welcome-модалки при повторном заходе.
    *   [x] Прогресс-бар считает только вопросы (без Intro/Process/Result).
*   [x] Главная страница (адаптивная под статус онбординга).
*   [x] Актуальный контекст темы (единая тема между разделами + кнопка «Новая тема»).

*   [x] Список готового контента (просмотр, фильтрация пустых).
*   [x] Свои промпты (CRUD).
*   [x] Интеграция оплаты AllPay:
    *   [x] Инициализация платежа (ветка payment_init).
    *   [x] Открытие в popup окне.
    *   [x] Обработка подтверждения (ветка payment_confirm).
    *   [x] PHP Webhook Proxy (`webhook-allpay.php`).
*   [x] Управление тарифами в БД (поля `tariff`, `tariff_expires_at`).
*   [x] Оптимизированный запрос профиля (свежий онбординг + статус тарифа за 1 запрос).
*   [x] Генераторы (каркасы):
    *   [x] Идея в пост
    *   [x] Сторис из текста
    *   [x] Заголовки

    *   [x] Reels
