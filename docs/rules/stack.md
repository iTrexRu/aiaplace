# Tech Stack

## Frontend
*   **Language**: JavaScript (ES6+).
*   **Framework**: None (Vanilla JS).
*   **Routing**: Custom (Hash-based or History API via `router.js`).
*   **Styles**: Pure CSS (Variables, Flexbox/Grid).

## Backend
*   **Platform**: n8n (Workflow Automation).
*   **Communication**: Webhooks (HTTP POST).
*   **Data Format**: JSON.

## Database
*   **System**: PostgreSQL.
*   **Access**: Through n8n nodes (Postgres Node).

## Infrastructure
*   **Deployment**: Static file serving + PHP (for simple dynamic needs like cache busting).
*   **Hosting**: Compatible with any PHP/Static host (e.g., Railway, DigitalOcean App Platform).
