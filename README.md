# Facont (AI Content Factory)

Веб-приложение для автоматизированного создания контента с помощью AI.

## Процесс работы (Деплой)

1.  **Локальная разработка**:
    *   Вносите изменения в код локально.
    *   Сохраняете файлы.

2.  **Деплой**:
    *   Делаете коммит и пуш в репозиторий GitHub.
    *   ```bash
        git add .
        git commit -m "Your message"
        git push origin main
        ```
    *   Настроен автодеплой на `facont.ai` (через GitHub Actions или Webhooks).

3.  **Установка**:
    *   Специальная установка не требуется. Проект работает как статический сайт с PHP-оберткой.

## Структура

*   `facont/` - Корневая папка приложения (Frontend).
    *   `app/` - Исходный код SPA (HTML части, JS модули, CSS).
    *   `index.php` - Точка входа (редирект на `landing.html`).
*   `_dev/` - Инструменты разработки и тема WordPress (устаревшие или вспомогательные).

## Конфигурация

*   API URL жестко закодирован в `facont/app/js/core/api.js`.
*   Бэкенд реализован на базе n8n (Webhook).
